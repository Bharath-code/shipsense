# Shipsense 🚀

**Know if your GitHub project is growing — and what to do today.**

Shipsense is an AI-powered growth intelligence tool for indie hackers, open-source maintainers, and devtool founders.  
It tracks your repository momentum and gives **clear daily actions** to help you maintain traction, attract contributors, and grow faster.

---

## ✨ Why Shipsense?

Most founders check:

- ⭐ Stars  
- 🐛 Issues  
- 🔀 Pull Requests  
- 📈 Traffic  
- 📦 Downloads  

… but still don’t know:

> **Is my project actually growing? What should I do today?**

Shipsense turns raw repo data into **actionable growth intelligence.**

## ⚙️ How It Works

1. Connect your GitHub account  
2. Select a repository  
3. Shipsense analyzes growth signals  
4. Receive daily intelligence + action suggestions  

Time to value: **< 30 seconds**

---

## 🧩 Tech Stack (MVP)

- **Frontend:** SvelteKit  
- **Backend / DB:** Supabase (Postgres)  
- **Data Source:** GitHub GraphQL + REST APIs  
- **AI Layer:** LLM-based insight generator  
- **Jobs / Scheduling:** Cron / Queue workers  
- **Email:** Resend / SMTP
